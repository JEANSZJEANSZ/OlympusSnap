# Fonts (bundled — no CDN / API)

## Press Start 2P
UI pixel face. Shipped via npm `@fontsource/press-start-2p` and imported in `src/main.js`.
SIL Open Font License — fine for booths.

## Greek-Freak
Display face for OLYMPUS / SNAP titles.
File: `Greek-Freak.woff` (local `@font-face` in `global.css`).

Previously from fonts.cdnfonts.com. Often personal-use / demo licensing —
for public or commercial booths, confirm / replace with a properly licensed webfont.
